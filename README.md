**Mathematics Quiz Game in C**
**Overview**

Mathematics Quiz Game is a console-based application developed in C programming language. The program generates random Mathematic questions and allows users to test their mathematical skills through an interactive quiz.

**Features**

- Addition Quiz
- Subtraction Quiz
- Multiplication Quiz
- Random Question Generation
- Instant Answer Validation
- Final Score Calculation
- Performance Feedback

**Technologies Used**

- C Programming Language
- Standard C Libraries
  - stdio.h
  - stdlib.h
  - time.h

**How to Run**

1. Compile the program:
   
   gcc main.c -o quiz

2. Run the executable:
   
   ./quiz
